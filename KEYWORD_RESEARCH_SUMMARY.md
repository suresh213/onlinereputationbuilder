- [ ] Primary keyword in H1 (once, naturally)
- [ ] Primary keyword in meta title (not stuffed)
- [ ] LSI keywords in H2s (related terms, not exact match)
- [ ] Question format in at least one H2 (for featured snippets)

### On-Page SEO
- [ ] Meta description includes primary keyword + benefit
- [ ] URL slug includes primary keyword
- [ ] Image alt text includes keyword variation
